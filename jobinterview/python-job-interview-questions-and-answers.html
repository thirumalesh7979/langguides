1. What sets a list apart from a tuple in Python?

Mutability: Lists are mutable, meaning their elements can be modified after creation, whereas tuples are immutable and cannot be changed once created.

Syntax: Lists are defined using square brackets [], while tuples use parentheses ().

Usage: Lists are commonly used for storing collections of items where the order and mutability are important, such as maintaining a dynamic set of elements. Tuples are typically used for grouping related values together, like returning multiple values from a function or representing an immutable collection of elements.

2. How can you eliminate duplicate elements from a list in Python?

To eliminate duplicate elements from a list in Python, you can convert the list to a set and then convert it back to a list. 


my_list = [1, 2, 3, 2, 4, 1, 5, 6, 4, 3]
unique_list = list(set(my_list))
print(unique_list) # [1, 2, 3, 4, 5, 6]

3. Explain the concept of list comprehension and its usage in Python.

List comprehension is a concise way to create lists in Python based on existing iterables (lists, tuples, strings, etc.) and optionally applying conditions or transformations to the elements. It allows you to combine looping, conditional statements, and value assignment into a single line of code.

The general syntax for list comprehension is: [expression for item in iterable if condition]


squares = [x**2 for x in range(1, 6)]
print(squares) # [1, 4, 9, 16, 25]

4. What role does the "self" keyword play in Python class definitions?

In Python class definitions, the self keyword is used to refer to the instance of the class. It represents the specific object created from that class and allows you to access its attributes and methods.

When defining methods within a class, the first parameter of each method should be self, which is a convention in Python. It's used to refer to the instance on which the method is being called. By convention, self is used, but you can use any name you prefer. 


class Person:
     def __init__(self, name):
          self.name = name

     def say_hello(self):
          print("Hello, my name is", self.name)

person = Person("Jane")
person.say_hello()
# Hello, my name is Jane

5. How do you handle exceptions in Python programming?

Exception handling in Python allows you to gracefully handle errors and exceptions that may occur during program execution. The try-except block is used to catch and handle exceptions. 


try:
     num1 = int(input("Enter a number: "))
     num2 = int(input("Enter another number: "))
     result = num1 / num2
     print("Result:", result)
except ZeroDivisionError:
     print("Error: Division by zero")
except ValueError:
     print("Error: Invalid input")

In this example, the try block attempts to perform division and print the result. If an exception is raised, such as ZeroDivisionError or ValueError, the corresponding except block is executed, providing an appropriate error message.

6. How do you read a text file line by line using Python?

To read a text file line by line in Python, you can use a combination of the open() function and a for loop.


with open('file.txt', 'r') as file:
for line in file:
     print(line)

In this example, the open() function is used to open the file named "file.txt" in read mode ('r'). The file is opened within a with statement, which ensures that the file is automatically closed after reading. The for loop iterates over each line in the file, and you can perform operations on each line inside the loop.

8. What distinguishes the usage of "==" and "is" in Python?

== (equality operator): It compares the values of two objects or variables and returns True if the values are equal, and False otherwise. It checks for equality based on the content of the objects.

is (identity operator): It checks if two objects or variables refer to the same memory location. It returns True if both objects point to the same memory location, and False otherwise. It checks for identity based on the object's memory address.

To summarize, == checks if the values of two objects are the same, while is checks if the objects themselves are the same.

9. How do you sort a dictionary based on its values in Python?

In Python, dictionaries are inherently unordered. However, you can sort a dictionary based on its values by using the sorted() function and a lambda function as the key parameter.


my_dict = {'apple': 3, 'banana': 1, 'orange': 2}
sorted_dict = dict(sorted(my_dict.items(), key=lambda x: x[1]))

print(sorted_dict)

In this example, the sorted() function is used to sort the dictionary my_dict based on its values. The items() method returns a list of key-value pairs, which is then sorted using the lambda function lambda x: x[1], which refers to the second element (value) of each pair. The sorted items are then converted back into a dictionary using the dict() constructor.

10. Shed light on the concept of generators and their application in Python.

Memory efficiency: Generators generate values one at a time, so they don't require a large amount of memory to store the entire sequence.

Laziness: Values are generated on-demand, making generators suitable for handling large or infinite sequences.

Iteration support: Generators can be iterated over using a for loop or by using them in other iterable contexts.

State retention: Generator functions can maintain their internal state between successive calls, allowing them to resume where they left off.

15. What are decorators in Python and how can they be implemented?

Decorators in Python are a way to modify the behavior of functions or classes without directly changing their source code. They allow you to wrap a function or a class with another function, adding extra functionality or modifying its behavior. 


def decorator_function(func):
     def wrapper():
          print("Before function execution")
          func()
          print("After function execution")
     return wrapper

@decorator_function
def my_function():
     print("Inside my_function")

my_function()

The @decorator_function syntax is used to apply the decorator to the my_function. When my_function() is called, the decorator function decorator_function is automatically executed, modifying the behavior of my_function by adding print statements before and after its execution.

16. How do you determine the most frequently occurring element in a list using Python?

To determine the most frequently occurring element in a list, you can use the collections.Counter class from the collections module.


from collections import Counter

my_list = [1, 2, 3, 2, 4, 3, 2, 1, 2, 4, 2]
counter = Counter(my_list)

most_common_element = counter.most_common(1)[0][0]

print("Most frequent element:", most_common_element)

In this example, the Counter class is used to create a counter object counter from the my_list. The most_common() method is then called on the counter object, specifying 1 as the argument to retrieve the most common element and its count as a list of tuples. By accessing [0][0], we obtain the most frequent element.

17. Explain the concept of lambda functions and their usage in Python.

Lambda functions have the following characteristics

They are written in a single line.

They can take any number of arguments but only have one expression.

They automatically return the result of the expression.


# Regular function
def square(x):
     return x ** 2

print(square(5))  # Output: 25

# Lambda function
square_lambda = lambda x: x ** 2

print(square_lambda(5))  # Output: 25

In this Python example, the square() function and square_lambda lambda function both calculate the square of a number. The lambda function is defined in a single line without a name and assigned to the variable square_lambda.

19. What distinguishes instance methods, class methods, and static methods in Python?

Instance methods: These methods are defined inside a class and operate on instances of that class. They have access to the instance itself through the self parameter. Instance methods can access and modify instance attributes and are typically used for actions or behaviors specific to individual instances.

Class methods: These methods are defined using the @classmethod decorator and operate on the class itself, rather than instances. They have access to the class through the cls parameter. Class methods can access class-level attributes and perform operations that affect the entire class. They are often used for alternative constructors or for modifying class attributes.

Static methods: These methods are defined using the @staticmethod decorator and do not have access to the instance or class. They behave like regular functions but are defined within a class for organizational purposes. Static methods are usually used for utility functions that don't require access to instance or class state. 


class MyClass:
     def instance_method(self):
          print("Instance method")

     @classmethod
     def class_method(cls):
          print("Class method")

     @staticmethod
     def static_method():
          print("Static method")
 
 # Instance method called on an instance
 my_obj = MyClass()
 my_obj.instance_method()
 
 # Class method called on the class itself
 MyClass.class_method()
 
 # Static method called without an instance or class
 MyClass.static_method()

20. How do you calculate the length of a string without utilizing the built-in len() function in Python?

To calculate the length of a string without using the built-in len() function, you can iterate over the characters in the string and count them manually. 


def calculate_length(string):
     count = 0
     for _ in string:
          count += 1
     return count
 
my_string = "Hello, world!"
length = calculate_length(my_string)

print("Length of the string:", length)

21. What is the purpose of the "pass" statement in Python programming?

The pass statement in Python is a placeholder statement that does nothing. It is used when a statement is syntactically required but you don't want to perform any action or provide any code at that point. It is commonly used as a placeholder for future code or in empty function or class definitions. The "pass" statement allows the code to pass over that point without causing a syntax error. 


def my_function():
     pass  # No action is performed

class MyClass:
     def my_method(self):
          pass  # No action is performed

In this example, both the "my_function" and "my_method" are defined, but they don't have any code inside them. By using the "pass" statement, the code is valid, and the interpreter will continue to the next line without raising an error.

25. What is the significance of the "super()" function in Python?

The super() function in Python is used to call a method or access an attribute from a parent class within a subclass. It is often used in inheritance when a subclass needs to extend or override the behavior of its parent class. 


class ParentClass:
     def __init__(self):
          self.attribute = "Parent attribute"

     def method(self):
          print("Parent method")

class ChildClass(ParentClass):
     def __init__(self):
          super().__init__()  # Call the parent class's __init__() method
          self.attribute = "Child attribute"  # Override the attribute

     def method(self):
          super().method()  # Call the parent class's method
          print("Child method")
 
# Create an instance of the child class
child_obj = ChildClass()

# Access the attribute and call the method
print(child_obj.attribute)
child_obj.method()

In this example, the "ChildClass" inherits from the "ParentClass". The "ChildClass" overrides the "init()" method and the "method()" method. By using "super().init()", the parent class's "init()" method is called to initialize the common attribute. Similarly, "super().method()" is used to call the parent class's "method()" and extend its behavior.

26. How do you iterate over multiple lists concurrently in Python?

To iterate over multiple lists concurrently in Python, you can use the zip() function. The zip() function takes multiple iterable objects as arguments and returns an iterator that aggregates elements from each iterable.


list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']
list3 = [True, False, True]

for item1, item2, item3 in zip(list1, list2, list3):
     print(item1, item2, item3)

# Output
# 1 a True
# 2 b False
# 3 c True

The zip() function stops iterating when the shortest input iterable is exhausted. If the lists have different lengths and you want to iterate until the longest list is exhausted, you can use the zip_longest() function from the itertools module.

27. What is the distinction between a shallow copy and a deep copy of an object in Python?

Shallow copy: A shallow copy creates a new object that references the original object's memory. It creates a new container object but shares the references to the objects contained within the original object. Any changes made to the original object's elements will also be reflected in the copied object's elements. Shallow copy can be performed using the copy() method or the copy module's copy() function.

Deep copy: A deep copy creates a new object and recursively copies the entire object hierarchy, including all nested objects. It creates completely independent copies of all objects, including the objects contained within the original object. Changes made to the original object or its elements will not affect the copied object. Deep copy can be performed using the copy module's deepcopy() function. 


import copy

# Original list
original_list = [1, [2, 3], 4]

# Shallow copy
shallow_copy = copy.copy(original_list)
shallow_copy[0] = 5
shallow_copy[1].append(6)

# Deep copy
deep_copy = copy.deepcopy(original_list)
deep_copy[0] = 7
deep_copy[1].append(8)

print("Original list:", original_list)
print("Shallow copy:", shallow_copy)
print("Deep copy:", deep_copy)
     
# Output 
# Original list: [1, [2, 3, 6], 4]
# Shallow copy: [5, [2, 3, 6], 4]
# Deep copy: [7, [2, 3, 8], 4]

30. How do you check if a key exists in a dictionary in Python?

To check if a key exists in a dictionary, you can use the "in" operator or the "get()" method. 


# Using the "in" operator

my_dict = {'a': 1, 'b': 2, 'c': 3}

if 'a' in my_dict:
     print("'a' key exists in the dictionary")
else:
     print("'a' key does not exist in the dictionary")



# Using the "get()" method

my_dict = {'a': 1, 'b': 2, 'c': 3}

if my_dict.get('a') is not None:
     print("'a' key exists in the dictionary")
else:
     print("'a' key does not exist in the dictionary")

Both approaches will give the same result


'a' key exists in the dictionary
