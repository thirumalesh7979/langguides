Python Lists

In Python, a list is a versatile and commonly used data structure that allows you to store and manipulate a collection of elements. Lists are mutable (can change), meaning you can modify their contents after they are created.

Creating a List

To create a list, you enclose a sequence of elements within square brackets [], separated by commas. Lists can contain elements of different data types, including numbers, strings, or even other lists. 


numbers = [1, 2, 3, 4, 5]
fruits = ['apple', 'banana', 'orange']
mixed_list = [10, 'hello', 3.14, True]

Accessing List Element

You can access individual elements in a list using indexing. Python uses zero-based indexing, so the first element has an index of 0. You can also use negative indices to access elements from the end of the list. 


numbers = [1, 2, 3, 4, 5]
print(numbers[0])  # Output: 1
print(numbers[-1])  # Output: 5

Modifying List Elements

Lists are mutable, so you can modify their elements by assigning new values to specific indices. 


numbers = [1, 2, 3, 4, 5]
numbers[2] = 10
print(numbers)  # Output: [1, 2, 10, 4, 5]

List Operations

Lists support various operations, such as concatenation (+), repetition (*), and slicing (:). 


list1 = [1, 2, 3]
list2 = [4, 5, 6]
result = list1 + list2
print(result)  # Output: [1, 2, 3, 4, 5, 6]

repeated = [1, 2] * 3
print(repeated)  # Output: [1, 2, 1, 2, 1, 2]

numbers = [1, 2, 3, 4, 5]
sliced = numbers[1:4]
print(sliced)  # Output: [2, 3, 4]

List Methods

Lists come with built-in methods that allow you to perform various operations on them. Some common methods include append(), pop(), insert(), remove(), sort(), and reverse(). 


numbers = [3, 1, 4, 2, 5]
numbers.append(6)
print(numbers)  # Output: [3, 1, 4, 2, 5, 6]

numbers.pop()
print(numbers)  # Output: [3, 1, 4, 2, 5]

numbers.insert(2, 7)
print(numbers)  # Output: [3, 1, 7, 4, 2, 5]

numbers.remove(4)
print(numbers)  # Output: [3, 1, 7, 2, 5]

numbers.sort()
print(numbers)  # Output: [1, 2, 3, 5, 7]

numbers.reverse()
print(numbers)  # Output: [7, 5, 3, 2, 1]
