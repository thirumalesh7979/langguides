Creating Objects in Python

In Python, objects are instances of classes. Once you have defined a class, you can create objects of that class using a process called instantiation

Class Definition

Before creating objects, you need to define a class. A class serves as a blueprint for creating objects with specific attributes and behaviors. 


class Car:
     def __init__(self, brand, year):
          self.brand = brand
          self.year = year

     def drive(self):
          print("The car is driving.")

In this example, we define a class named Car with an __init__() method as the constructor and a drive() method.

Instantiation

To create an object (instance) of a class, you call the class name followed by parentheses. This invokes the constructor to initialize the object. 


car1 = Car("Toyota", 2022)
car2 = Car("Honda", 2021)

In this code, we create two car objects, car1 and car2, by calling the Car class and passing the required arguments to the constructor.

Accessing Object Attributes and Methods

Once you have created objects, you can access their attributes and call their methods using the dot notation. 


print(car1.brand)    # Output: Toyota
print(car2.year)     # Output: 2021
car1.drive()         # Output: The car is driving.

This code accesses the brand attribute of car1, the year attribute of car2, and calls the drive() method of car1.

Object Identity

Each object in Python has a unique identity, which can be obtained using the id() function. The id() function returns a unique identifier for the object. 


print(id(car1)) # Output: 140176244892944
print(id(car2)) # Output: 140176244893200
