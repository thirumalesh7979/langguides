Python Tuples

In Python, a tuple is an ordered, immutable collection of elements. Tuples are similar to lists, but unlike lists, they cannot be modified once created. Tuples are often used to group related data together.

Creating a Tuple

To create a tuple, you enclose a sequence of elements within parentheses (), separated by commas. Although parentheses are optional, it is a good practice to use them to make the tuple more explicit. 


my_tuple = (1, 2, 3)
fruits_tuple = ('apple', 'banana', 'orange')
mixed_tuple = (10, 'hello', 3.14, True)

Accessing Tuple Elements

You can access individual elements in a tuple using indexing, similar to lists. The indexing starts from 0, and negative indices can be used to access elements from the end of the tuple. 


my_tuple = (1, 2, 3)
print(my_tuple[0])  # Output: 1
print(my_tuple[-1])  # Output: 3

Tuple Packing and Unpacking

Tuple packing is the process of creating a tuple by assigning multiple values to a single variable, while tuple unpacking allows you to assign individual values from a tuple to multiple variables. 


# Tuple packing
my_tuple = 1, 2, 3
print(my_tuple)  # Output: (1, 2, 3)

# Tuple unpacking
a, b, c = my_tuple
print(a)  # Output: 1
print(b)  # Output: 2
print(c)  # Output: 3

Tuple Operations

Tuples support several operations, such as concatenation (+) and repetition (*). However, since tuples are immutable, operations that modify the tuple directly, like appending or removing elements, are not supported. 


tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)
result = tuple1 + tuple2
print(result)  # Output: (1, 2, 3, 4, 5, 6)

repeated = (1, 2) * 3
print(repeated)  # Output: (1, 2, 1, 2, 1, 2)

Tuple Methods

Tuples have fewer built-in methods compared to lists since they are immutable. However, common methods such as count() and index() are available for tuple operations. 


my_tuple = (1, 2, 3, 2)
count = my_tuple.count(2)
print(count)  # Output: 2

index = my_tuple.index(3)
print(index)  # Output: 2
