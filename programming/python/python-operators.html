Python Operators and Expressions

Operators in Python are symbols or special keywords that perform operations on operands (values or variables). Expressions, on the other hand, are combinations of operators, operands, and other elements that produce a value.

1. Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations.


x = 10
y = 3

addition = x + y
print(addition)  # Output: 13

subtraction = x - y
print(subtraction)  # Output: 7

multiplication = x * y
print(multiplication)  # Output: 30

division = x / y
print(division)  # Output: 3.3333333333333335

modulo = x % y
print(modulo)  # Output: 1

exponentiation = x ** y
print(exponentiation)  # Output: 1000

2. Comparison Operators

Comparison operators are used to compare values and return a Boolean result (True or False). 


x = 10
y = 3

greater_than = x > y
print(greater_than)  # Output: True

less_than = x < y
print(less_than)  # Output: False

equal_to = x == y
print(equal_to)  # Output: False

not_equal_to = x != y
print(not_equal_to)  # Output: True

greater_than_or_equal_to = x >= y
print(greater_than_or_equal_to)  # Output: True

less_than_or_equal_to = x <= y
print(less_than_or_equal_to)  # Output: False

3. Logical Operators

Logical operators are used to combine and manipulate Boolean values. 


x = True
y = False

logical_and = x and y
print(logical_and)  # Output: False

logical_or = x or y
print(logical_or)  # Output: True

logical_not_x = not x
print(logical_not_x)  # Output: False

logical_not_y = not y
print(logical_not_y)  # Output: True

4. Assignment Operators

Assignment operators are used to assign values to variables.


x = 10
print(x)  # Output: 10

x += 5  # Equivalent to x = x + 5
print(x)  # Output: 15

x -= 3  # Equivalent to x = x - 3
print(x)  # Output: 12

x *= 2  # Equivalent to x = x * 2
print(x)  # Output: 24

x /= 4  # Equivalent to x = x / 4
print(x)  # Output: 6.0

x %= 5  # Equivalent to x = x % 5
print(x)  # Output: 1.0

x **= 3  # Equivalent to x = x ** 3
print(x)  # Output: 1.0

5. Bitwise Operators

Bitwise operators perform operations on binary representations of numbers.


x = 5  # Binary: 101
y = 3  # Binary: 011

bitwise_and = x & y  # Binary: 001
print(bitwise_and)  # Output: 1

bitwise_or = x | y  # Binary: 111
print(bitwise_or)  # Output: 7

bitwise_xor = x ^ y  # Binary: 110
print(bitwise_xor)  # Output: 6

bitwise_not_x = ~x  # Binary: 11111111111111111111111111111010 (two's complement)
print(bitwise_not_x)  # Output: -6

bitwise_left_shift = x << 1  # Binary: 1010
print(bitwise_left_shift)  # Output: 10

bitwise_right_shift = x >> 1  # Binary: 10
print(bitwise_right_shift)  # Output: 2

6. Membership Operators

Membership operators are used to check if a value exists within a sequence. 


fruits = ['apple', 'banana', 'cherry']

is_apple_present = 'apple' in fruits
print(is_apple_present)  # Output: True

is_mango_present = 'mango' in fruits
print(is_mango_present)  # Output: False

is_grape_absent = 'grape' not in fruits
print(is_grape_absent)  # Output: True

7. Identity Operators

Identity operators are used to compare the memory addresses of objects. 


x = 10
y = 10
z = [1, 2, 3]

are_x_and_y_same = x is y
print(are_x_and_y_same)  # Output: True

are_x_and_z_same = x is z
print(are_x_and_z_same)  # Output: False

are_x_and_z_different = x is not z
print(are_x_and_z_different)  # Output: True
