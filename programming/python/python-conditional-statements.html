Python Conditional Statements

Conditional statements in Python allow you to execute different blocks of code based on certain conditions. The three main components of conditional statements are the "if" statement, the "elif" (short for "else if") statement, and the "else" statement.

if Statement

The "if" statement is used to execute a block of code if a specific condition is true. 


age = 18

if age >= 18:
     print("You are eligible to vote.") # You are eligible to vote.

if-else Statement

The "if-else" statement is used to execute one block of code if a condition is true, and another block of code if the condition is false. 


age = 15

if age >= 18:
     print("You are eligible to vote.")
else:
     print("You are not eligible to vote.") # You are not eligible to vote.

if-elif-else Statement

The "if-elif-else" statement is used when you have multiple conditions to check. It allows you to execute different blocks of code based on the conditions. 


score = 85

if score >= 90:
     print("You scored an A.")
elif score >= 80:
     print("You scored a B.") # You scored a B.
elif score >= 70:
     print("You scored a C.")
elif score >= 60:
     print("You scored a D.")
else:
     print("You scored an F.")

It's important to note that in an if-elif-else structure, once a condition evaluates to True and its corresponding block of code is executed, the program exits the structure and does not evaluate any subsequent conditions.

Nested if Statements

You can also nest conditional statements inside one another to handle more complex conditions.


age = 20
grade = 'B'

if age >= 18:
     if grade == 'A':
          print("You are eligible for a scholarship.")
     else:
          print("You are eligible to apply.") # You are eligible to apply.
else:
     print("You are not eligible.")

Ternary Operator

Python also provides a shorthand notation called the ternary operator for writing simple if-else statements in a single line. 


age = 20

result = "You are eligible to vote." if age >= 18 else "You are not eligible to vote."
print(result) # You are eligible to vote.
