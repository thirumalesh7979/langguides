Importing and Using Modules in Python

In Python, modules are files containing Python code that define functions, classes, and variables that can be used in other programs. Modules allow you to organize your code into separate files, promote code reusability, and avoid naming conflicts.

Importing Modules

To import a module in Python, you can use the import statement followed by the name of the module. 

import math

In this example, the math module is imported, which provides various mathematical functions and constants.

Using Functions and Variables from a Module

Once a module is imported, you can access its functions, classes, and variables using the dot notation. 


import math

print(math.sqrt(25))  # Output: 5.0
print(math.pi)  # Output: 3.141592653589793

In this code, the sqrt() function from the math module is called to calculate the square root of 25. The constant pi from the math module is also accessed and printed.

Importing Specific Functions or Variables from a Module

Instead of importing the entire module, you can import specific functions or variables using the from keyword. 


from math import sqrt, pi

print(sqrt(25))  # Output: 5.0
print(pi)  # Output: 3.141592653589793

Importing a Module with an Alias

You can import a module with an alias (alternate name) using the as keyword. This can be useful to provide a shorter or more descriptive name for the module. 


import math as m

print(m.sqrt(25))  # Output: 5.0

Creating a Custom Module

To create a module, you need to create a new Python file with a .py extension. This file will serve as your module and can contain any valid Python code. 


# my_module.py file
def greet(name):
     print(f"Hello, {name}!")

def square(n):
     return n * n

PI = 3.141592653589793

In this example, the my_module.py file defines two functions (greet() and square()) and a variable (PI).

Importing Your Module

Once you've created your module, you can import it into other Python programs (like we discussed earlier) using the import statement. 


import my_module

my_module.greet("Radha")  # Output: Hello, Radha!
print(my_module.square(5))  # Output: 25
print(my_module.PI)  # Output: 3.141592653589793
