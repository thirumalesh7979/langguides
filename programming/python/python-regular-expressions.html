Python Regular Expressions

Regular expressions, also known as regex, are powerful tools for pattern matching and manipulation of strings in Python. The re module in Python provides functions and methods for working with regular expressions.

Importing the re Module

Before using regular expressions, you need to import the re module.

import re

Matching Patterns

The re module provides several functions for matching patterns in strings. The most commonly used function is re.match(), which attempts to match a pattern at the beginning of a string. 


import re

pattern = r"ab"
string = "abcdef"

match = re.match(pattern, string)
if match:
     print("Match found!") # Match found!
else:
     print("No match found!")

In this code, the pattern ab is matched against the string abcdef. If a match is found at the beginning of the string, it prints "Match found!".

Searching for Patterns

Another useful function is re.search(), which searches for a pattern anywhere in a string and returns the first match found. 


import re

pattern = r"cd"
string = "abcdef"

match = re.search(pattern, string)
if match:
     print("Match found!") # Match found!
else:
     print("No match found!")

In this Python code, the pattern cd is searched within the string abcdef. If a match is found anywhere in the string, it prints "Match found!".

Pattern Substitution

The re.sub() function allows you to substitute occurrences of a pattern in a string with a specified replacement. 


import re

pattern = r"blue"
string = "The sky is blue."

new_string = re.sub(pattern, "red", string) # The sky is red.
print(new_string)

In this code, the pattern blue is substituted with the string "red" in the given string. The resulting string is then printed.

Splitting Strings using Patterns

The re.split() function allows you to split a string based on a specified pattern. 


import re

pattern = r"\s+"  # Split on one or more whitespace characters
string = "Hello   World!"

parts = re.split(pattern, string)
print(parts) # ['Hello', 'World!']

In this code, the pattern \s+ matches one or more whitespace characters. The string is split into parts based on this pattern, and the resulting parts are printed.

An example of how to use regular expressions in Python to find email addresses within a given text


import re

def find_emails(text):
     pattern = r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}\b'
     emails = re.findall(pattern, text)
     return emails

# Example usage
text = "Contact us at info@example.com or support@example.com"
found_emails = find_emails(text)

if found_emails:
     print("Email addresses found:")
     for email in found_emails:
          print(email)
else:
     print("No email addresses found.")

# Output
# Email addresses found:
# info@example.com
# support@example.com

In this code, the find_emails function takes a string text as input and uses the re.findall() function to find all occurrences of email addresses within the text. The regular expression pattern r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}\b' is used to match email addresses.

The pattern breakdown

\b is a word boundary to ensure the email address is not part of a larger word.

[A-Za-z0-9._%+-]+ matches one or more alphanumeric characters, dots, underscores, percentage signs, plus signs, or hyphens for the username part.

@ matches the literal "@" symbol.

[A-Za-z0-9.-]+ matches one or more alphanumeric characters, dots, or hyphens for the domain name part before the top-level domain.

\. matches the literal dot before the top-level domain.

[A-Za-z]{2,} matches two or more alphabetical characters for the top-level domain.

\b is another word boundary to ensure the email address is not part of a larger word.
