Python Comments

What are the Python Comments?

Comments in Python are lines of code that are not executed by the interpreter. They are used to add explanatory notes or annotations to the code, making it easier for developers to understand and maintain the program.

There are two types of comments in Python

1. Single-line comments: These comments begin with a hash symbol (#) and continue until the end of the line. Any text after the hash symbol is considered a comment. 


# This is a single-line comment
print("Hello!")  # This is another single-line comment

2. Multi-line comments: Python doesn't have a specific syntax for multi-line comments like some other programming languages. However, you can use triple quotes (either single or double) to achieve a similar effect. When triple quotes are used as a comment, they can span multiple lines.


"""
This is a multi-line comment.
It can span across multiple lines.
"""

'''
This is another way to create a multi-line comment.
'''

print("Hello!")
