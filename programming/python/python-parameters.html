Python Function Parameters and Return Values

Function parameters and return values in Python allow you to pass data into functions and retrieve results from them. This tutorial will cover how to work with function parameters and return values

Function Parameters

Parameters are variables that are defined in the function's signature and used to accept input values when the function is called. They specify what data the function needs to perform its task. 


def greet(name):
     print(f"Hello, {name}!")

def add_numbers(a, b):
     sum = a + b
     print(f"The sum is: {sum}")

Positional Arguments

When calling a function, you can provide arguments based on their position in the function's parameter list. The order of the arguments should match the order of the parameters. 


greet("Radha")  # Output: Hello, Radha!

add_numbers(5, 3)  # Output: The sum is: 8

Keyword Arguments

Keyword arguments allow you to specify arguments by their parameter names when calling a function. This way, the order of the arguments doesn't matter. 


greet(name="Krishna")  # Output: Hello, Bob!

add_numbers(b=2, a=7)  # Output: The sum is: 9

Default Parameters

You can assign default values to function parameters. If an argument is not provided when calling the function, the default value will be used.


def greet_with_message(name, message="Hello"):
     print(f"{message}, {name}!")
 
 greet_with_message("Alice")  # Output: Hello, Alice!
 greet_with_message("Bob", "Hi")  # Output: Hi, Bob!

Return Values

Functions can return a value using the return statement. The returned value can be assigned to a variable or used in other parts of the program. 


def multiply_numbers(a, b):
     product = a * b
     return product

result = multiply_numbers(4, 6)
print(result)  # Output: 24

Multiple Return Values

Functions can also return multiple values by separating them with commas in the return statement. These values can be captured in multiple variables when calling the function. 


def calculate_rectangle_properties(length, width):
     area = length * width
     perimeter = 2 * (length + width)
     return area, perimeter

area, perimeter = calculate_rectangle_properties(4, 5)
print(area)  # Output: 20
print(perimeter)  # Output: 18
