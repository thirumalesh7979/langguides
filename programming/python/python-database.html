Working with databases (MongoDB, MySQL) in Python

Python provides libraries and modules that make it easy to work with databases. Two popular databases, MongoDB and MySQL, have specific libraries available that enable you to interact with them using Python.

MongoDB

MongoDB is a popular NoSQL database that stores data in flexible, JSON-like documents. To work with MongoDB in Python, you can use the PyMongo library. 

+
# Example code for connecting to a MongoDB database and performing basic operations

from pymongo import MongoClient

# Connect to MongoDB
client = MongoClient("mongodb://localhost:27017")

# Access a database
db = client["mydatabase"]

# Access a collection (equivalent to a table in SQL)
collection = db["mycollection"]

# Insert a document
document = {"name": "John", "age": 30}
inserted_document = collection.insert_one(document)
print("Inserted document ID:", inserted_document.inserted_id)

# Find documents
documents = collection.find({"age": {"$gt": 25}})
for doc in documents:
     print(doc)

# Update a document
filter = {"name": "John"}
update = {"$set": {"age": 35}}
updated_document = collection.update_one(filter, update)
print("Modified document count:", updated_document.modified_count)

# Delete a document
delete_result = collection.delete_one({"name": "John"})
print("Deleted document count:", delete_result.deleted_count)

# Disconnect from MongoDB
client.close()

In this code, the PyMongo library is used to establish a connection to a MongoDB server, access a specific database, and perform operations such as insertion, querying, updating, and deletion.

MySQL

MySQL is a popular relational database management system. To work with MySQL databases in Python, you can use the mysql-connector-python library. 


import mysql.connector

# Connect to MySQL
conn = mysql.connector.connect(
     host="localhost",
     user="yourusername",
     password="yourpassword",
     database="mydatabase"
)

# Create a cursor object to execute SQL queries
cursor = conn.cursor()

# Execute SQL queries
cursor.execute("SELECT * FROM mytable")
rows = cursor.fetchall()
for row in rows:
     print(row)

# Insert a record
sql = "INSERT INTO mytable (name, age) VALUES (%s, %s)"
values = ("John", 30)
cursor.execute(sql, values)
conn.commit()
print("Inserted record ID:", cursor.lastrowid)

# Update a record
sql = "UPDATE mytable SET age = %s WHERE name = %s"
values = (35, "John")
cursor.execute(sql, values)
conn.commit()
print("Modified record count:", cursor.rowcount)

# Delete a record
sql = "DELETE FROM mytable WHERE name = %s"
values = ("John",)
cursor.execute(sql, values)
conn.commit()
print("Deleted record count:", cursor.rowcount)

# Disconnect from MySQL
cursor.close()
conn.close()

In this code, the mysql-connector-python library is used to establish a connection to a MySQL database, create a cursor object for executing SQL queries, and perform operations such as selecting, inserting, updating, and deleting records.
