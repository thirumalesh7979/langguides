Python Datatypes

Data types in Python are used to represent different kinds of data and enable various operations on them. Python provides several built-in data types, including numeric types, strings, booleans, lists, tuples, sets, and dictionaries.

1. Numeric Types

Python supports two main numeric types: integers and floating-point numbers. 


# Integer
age = 23
print(age)  # Output: 23

# Float
temperature = 99.6
print(temperature)  # Output: 99.6

2. String Type

Strings in Python are used to represent a sequence of characters. They can be enclosed in either single quotes (' ') or double quotes (" "). 


name = "Radha"
print(name)  # Output: Radha

greeting = 'Hello!'
print(greeting)  # Output: Hello!

3. Boolean Type

The boolean data type represents the truth values True and False. Booleans are commonly used in control structures and logical operations. 


is_valid = True
print(is_valid)  # Output: True

has_permission = False
print(has_permission)  # Output: False

4. List Type

Lists are ordered collections of items, enclosed in square brackets ([ ]). They can contain elements of different data types. 


fruits = ['apple', 'banana', 'cherry']
print(fruits)  # Output: ['apple', 'banana', 'cherry']

numbers = [1, 2, 3, 4, 5]
print(numbers)  # Output: [1, 2, 3, 4, 5]

5. Tuple Type

Lists are ordered collections of items, enclosed in square brackets ([ ]). They can contain elements of different data types. 


coordinates = (10, 20)
print(coordinates)  # Output: (10, 20)

colors = ('red', 'green', 'blue')
print(colors)  # Output: ('red', 'green', 'blue')

6. Set Type

Sets are unordered collections of unique elements, enclosed in curly braces ({ }). They are useful for tasks like removing duplicates and performing mathematical set operations. 


numbers = {1, 2, 3, 4, 4, 5}
print(numbers)  # Output: {1, 2, 3, 4, 5}

vowels = {'a', 'e', 'i', 'o', 'u'}
print(vowels)  # Output: {'a', 'e', 'i', 'o', 'u'}

7. Dictionary Type

Dictionaries store key-value pairs. They are unordered, mutable, and enclosed in curly braces ({ }). Each key is unique within a dictionary. 


person = {'name': 'Radha', 'age': 25, 'city': 'New York'}
print(person)  # Output: {'name': 'Radha', 'age': 25, 'city': 'New York'}

car = {'brand': 'Ford', 'model': 'Mustang', 'year': 2022}
print(car)  # Output: {'brand': 'Ford', 'model': 'Mustang', 'year': 2022}
