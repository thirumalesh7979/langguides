Writing to Files in Python

Writing data to files is a common task in many programs. Python provides built-in functions and methods to facilitate file output.

Opening a File

To write to a file, you first need to open it using the open() function. The open() function takes the file path as a parameter and returns a file object that can be used to write data to the file. 

file = open('example.txt', 'w')

In the example above, example.txt is the name of the file we want to write to, and 'w' specifies that we want to open the file in write mode. If the file does not exist, it will be created. If it already exists, the existing contents will be overwritten.

Writing to a File

Once the file is opened, you can write data to it using the write() method of the file object. The write() method takes a string as a parameter and writes it to the file. 

file = open('example.txt', 'w')
file.write('Hello, World!\n')
file.write('This is a new line.')
file.close()

This code writes the string 'Hello, World!' followed by a newline character ('\n'), and then writes the string 'This is a new line.' to the file example.txt. After writing the data, it's important to close the file using the close() method to ensure that the changes are saved.

Appending to a File

If you want to add content to an existing file without overwriting the existing contents, you can open the file in append mode by specifying 'a' instead of 'w' when opening the file. 

file = open('example.txt', 'a')
file.write('This is additional content.')
file.close()

Using the with Statement

It's recommended to use the with statement when writing to files in Python. The with statement automatically takes care of closing the file, even if an exception occurs. 


with open('example.txt', 'w') as file:
	file.write('Hello, World!')

It's important to note that when writing to files, the data is initially stored in a buffer, and it may not be immediately written to the file on disk. To ensure that the data is written to the file immediately, you can use the flush() method of the file object or close the file using the close() method. 


file = open('example.txt', 'w')
file.write('Hello, World!')
file.flush()
