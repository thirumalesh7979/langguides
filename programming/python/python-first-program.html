First Python Program

To write and run a Python program, follow these steps:

1. Install Python

First, ensure that Python is installed on your system. You can download the latest version of Python from the official website (python.org) and follow the installation instructions for your operating system.

2. Choose a Text Editor or IDE

You can write Python code in any text editor, such as Notepad or Sublime Text. However, using an Integrated Development Environment (IDE) like PyCharm, Visual Studio Code, or IDLE provides a more convenient and feature-rich development environment.

3. Create a Python File

Open your preferred text editor or IDE and create a new file with a .py extension (e.g., my_program.py). This extension indicates that the file contains Python code.

4. Write Your Python Program

In the file, you can start writing your Python program. For example, let's write a simple program that prints "Welcome To the LangGuides!" to the console

print("Hello, World!")

After writing the code, save the file in a directory of your choice.

5. Run the Python Program

Open a command prompt or terminal and navigate to the directory where you saved your Python file. Use the cd command (change directory) to navigate to the desired directory. Then, execute the Python file by running the following command

python my_program.py

Make sure to replace my_program.py with the actual name of your Python file. Press Enter, and you will see the output of your Python program displayed in the command prompt or terminal

Welcome To the LangGuides!

Note: If you are using an IDE like PyCharm or Visual Studio Code, you can create a new Python project, write your code in the provided editor, and run the program from within the IDE itself. IDEs often offer additional features like code highlighting, debugging tools, and project management, which can enhance your development experience.

Python Indentation

In Python, indentation plays a crucial role in defining the structure and scope of code blocks. Unlike many other programming languages that use braces or keywords to denote code blocks, Python uses indentation.

In Python, the recommended standard for indentation is to use four spaces. This is the most commonly used and widely accepted convention within the Python community.

Here's an example to illustrate Python indentation:


if condition:
     # Indented code block
     statement_1
     statement_2
else:
     # Indented code block
     statement_3
     statement_4
     if another_condition:
          # Nested indented code block
          statement_5
