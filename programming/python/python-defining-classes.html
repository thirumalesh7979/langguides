Defining Classes in Python

In Python, classes are used to create objects and define their properties and behaviors.

Class Declaration

To define a class, you use the class keyword followed by the name of the class. By convention, class names start with an uppercase letter. 


class Car:
     pass

In this example, we define a class named Car. The pass statement is a placeholder that indicates an empty class. You can add attributes and methods inside the class definition.

Class Attributes

Class attributes are variables that are shared among all instances (objects) of a class. They are defined within the class but outside any methods. Class attributes are accessed using the class name. 


class Car:
     wheels = 4
     color = 'red'

In this Python code, we define two class attributes wheels and color for the Car class. These attributes are shared by all instances of the Car class.

Instance Attributes

Instance attributes are unique to each instance of a class. They are defined inside the class's methods using the self parameter. Each instance can have different values for instance attributes. 


class Car:
     def __init__(self, brand, year):
          self.brand = brand
          self.year = year

In this example, we define an __init__() method, which is a special method called the constructor. It initializes the instance attributes brand and year based on the arguments passed when creating an object.

Class Methods

Class methods are functions defined inside a class and can be called on the class itself, rather than on instances of the class. Class methods are defined using the @classmethod decorator. 


class Car:
     @classmethod
     def honk(cls):
          print("Beep beep!")

In this code, we define a class method honk() using the @classmethod decorator. The cls parameter refers to the class itself, and it is conventionally named cls.

Instance Methods

Instance methods are functions defined inside a class that operate on instances of the class. They are defined without any special decorators and automatically receive the self parameter, which refers to the instance. 


class Car:
     def __init__(self, brand):
          self.brand = brand

     def drive(self):
          print(f"Driving the {self.brand}")

In this example, we define an __init__() method as the constructor and an instance method drive(). The drive() method can be called on each individual car object.

Example


class Person:
     def __init__(self, name, age):
          self.name = name
          self.age = age

     def greet(self):
          print("Hello, my name is", self.name)

In this example, we define a class named Person. It has two attributes (name and age) and one method (greet). The __init__() method is a special method called the constructor, which is executed when creating an object of the class.
