Python Reading Files

Reading files is a common operation in many programs, and Python provides built-in functions and methods to facilitate file input.

Opening a File

To read a file, you first need to open it using the open() function. The open() function takes the file path as a parameter and returns a file object that can be used to access the file's contents. 

file = open('example.txt', 'r')

Reading the Entire File

Once the file is opened, you can read its contents using the read() method of the file object. The read() method returns the entire contents of the file as a string. 

file = open('example.txt', 'r')
contents = file.read()
print(contents)

Reading Line by Line

If you want to read a file line by line, you can use the readline() method of the file object. The readline() method reads and returns the next line of the file as a string. 


file = open('example.txt', 'r')
line = file.readline()
while line:
     print(line)
     line = file.readline()

Reading All Lines at Once

Another way to read a file line by line is by using a for loop with the file object. The file object behaves as an iterator, yielding one line at a time. 


file = open('example.txt', 'r')
for line in file:
     print(line)

Closing the File

After you finish reading the file, it's important to close it using the close() method of the file object. Closing the file releases system resources and ensures that any changes made to the file are saved. 


file = open('example.txt', 'r')
# Read the file contents
file.close()

It's recommended to use the with statement when reading files in Python. The with statement automatically takes care of closing the file, even if an exception occurs. 


with open('example.txt', 'r') as file:
     contents = file.read()
     print(contents)
