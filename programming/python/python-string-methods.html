Python String Methods

Python provides numerous built-in methods for strings to perform operations like splitting, replacing, finding substrings, and more. Some commonly used string methods include split(), replace(), find(), startswith(), endswith(), and strip().

1. capitalize(): Converts the first character of the string to uppercase and the rest to lowercase.

2. casefold(): Returns a lowercase version of the string. Similar to lower(), but handles more Unicode characters.

3. center(width, fillchar): Returns a centered string by padding it with the specified fillchar on both sides.

4. count(substring, start, end): Returns the number of occurrences of the substring in the string.

5. encode(encoding, errors): Returns an encoded version of the string using the specified encoding.

6. endswith(suffix, start, end) Checks if the string ends with the specified suffix.

7. expandtabs(tabsize) Returns a string where the tab characters \t are replaced by spaces.

8. find(substring, start, end): Searches for the substring in the string and returns the lowest index, or -1 if not found.

9. format(*args, **kwargs): Formats the string by substituting placeholders with values.

10. index(substring, start, end): Searches for the substring in the string and returns the lowest index, or raises a ValueError if not found.

11. isalnum(): Checks if all characters in the string are alphanumeric (letters or digits).

12. isalpha(): Checks if all characters in the string are alphabetic (letters).

13. isdecimal(): Checks if all characters in the string are decimal characters.

14. isdigit(): Checks if all characters in the string are digits.

def string_operations(string):
     # Check if the string starts with 'Hello'
     if string.startswith('Hello'):
         print("The string starts with 'Hello'")
     else:
         print("The string does not start with 'Hello'")
 
     # Count the number of occurrences of 'o' in the string
     count = string.count('o')
     print("The number of 'o' characters in the string:", count)
 
     # Replace 'World' with 'Python'
     replaced_string = string.replace('World', 'Python')
     print("Replaced string:", replaced_string)
 
     # Split the string into a list of words
     words = string.split(' ')
     print("Words in the string:", words)
 
     # Convert the string to uppercase
     uppercase_string = string.upper()
     print("Uppercase string:", uppercase_string)
 
     # Check if the string is numeric
     if string.isnumeric():
         print("The string is numeric")
     else:
         print("The string is not numeric")
 
 
# Test the program with an input string
input_string = "Hello, World! Welcome to Python"
string_operations(input_string)

Output


The string starts with 'Hello'
The number of 'o' characters in the string: 2
Replaced string: Hello, Python! Welcome to Python
Words in the string: ['Hello,', 'World!', 'Welcome', 'to', 'Python']
Uppercase string: HELLO, WORLD! WELCOME TO PYTHON
The string is not numeric
