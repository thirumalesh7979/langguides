Inheritance and polymorphism

Inheritance and polymorphism are two important concepts in object-oriented programming that allow for code reuse and flexibility.

Inheritance

Inheritance is the process of creating a new class (subclass) by deriving from an existing class (superclass). The subclass inherits the attributes and methods of the superclass and can add its own unique attributes and methods. Inheritance is useful for creating specialized versions of a class or extending its functionality. 


class Animal:
     def sound(self):
          print("The animal makes a sound.")

class Dog(Animal):
     def sound(self):
          print("The dog barks.")

In this example, we define a superclass Animal with a sound() method and a subclass Dog that inherits from Animal. The Dog class overrides the sound() method to provide a different implementation.

Polymorphism

Polymorphism allows objects of different classes to be treated as objects of a common superclass. This means that a variable of the superclass type can refer to objects of either the superclass or its subclasses. Polymorphism enables code to work with different types of objects through a common interface.


def make_sound(animal):
animal.sound()

animal1 = Animal()
animal2 = Dog()

make_sound(animal1)  # Output: The animal makes a sound.
make_sound(animal2)  # Output: The dog barks.

In this code, the make_sound() function takes an object of the superclass Animal as an argument. The function can accept both Animal objects and Dog objects because Dog is a subclass of Animal. Polymorphism allows the function to call the appropriate sound() method based on the actual type of the object.

Method Overriding

Method overriding occurs when a subclass provides its own implementation of a method inherited from the superclass. This allows the subclass to modify or extend the behavior of the inherited method. 


class Animal:
     def sound(self):
          print("The animal makes a sound.")

class Cat(Animal):
     def sound(self):
          print("The cat meows.")

In this example, the Cat class overrides the sound() method inherited from Animal to provide a different sound. When calling sound() on a Cat object, the overridden method in the Cat class is executed.
