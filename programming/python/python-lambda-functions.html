Lambda Functions in Python

Lambda functions, also known as anonymous functions, are a way to create small, one-line functions without explicitly defining them using the def keyword. Lambda functions are typically used for simple operations or as arguments to higher-order functions.

Syntax: 

lambda arguments: expression

Creating a Lambda Function

Lambda functions are created using the lambda keyword, followed by a list of arguments, a colon, and an expression. 

add = lambda a, b: a + b

Using Lambda Functions

Lambda functions are typically used in situations where a small, throwaway function is needed. They are often used as arguments to higher-order functions like map(), filter(), or reduce(), or in situations where a function needs to be defined quickly and concisely. 


# Using lambda with map()
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # Output: [1, 4, 9, 16, 25]

# Using lambda with filter()
numbers = [1, 2, 3, 4, 5]
even = list(filter(lambda x: x % 2 == 0, numbers))
print(even)  # Output: [2, 4]

# Using lambda with reduce()
from functools import reduce
numbers = [1, 2, 3, 4, 5]
product = reduce(lambda x, y: x * y, numbers)
print(product)  # Output: 120

Lambda functions are concise and allow for the creation of simple functions on the fly. However, they have some limitations. Lambda functions can only contain a single expression, and they cannot include statements or complex logic. If you find that the function logic becomes more complex, it is recommended to use a regular named function instead.
