Python Sets

In Python, a set is an unordered collection of unique elements. Sets are useful when you want to store a collection of distinct items and perform operations such as membership testing, intersection, union, and difference.

Creating a Set

To create a set, you can enclose a sequence of elements within curly braces {} or use the set() function. Duplicate elements are automatically removed, and the remaining unique elements are stored in the set. 

my_set = {1, 2, 3, 4, 5}
another_set = set([3, 4, 5, 6, 7])

Set Operations

Sets support various operations such as union, intersection, difference, and symmetric difference. These operations allow you to combine or compare sets and perform set-specific operations. 


set1 = {1, 2, 3, 4, 5}
set2 = {4, 5, 6, 7, 8}

union_set = set1.union(set2)
print(union_set)             # Output: {1, 2, 3, 4, 5, 6, 7, 8}

intersection_set = set1.intersection(set2)
print(intersection_set)      # Output: {4, 5}

difference_set = set1.difference(set2)
print(difference_set)        # Output: {1, 2, 3}

symmetric_difference_set = set1.symmetric_difference(set2)
print(symmetric_difference_set)  # Output: {1, 2, 3, 6, 7, 8}

Modifying a Set

Sets are mutable, so you can add and remove elements from a set using methods like add(), remove(), and discard(). 


my_set = {1, 2, 3}

my_set.add(4)
print(my_set)                # Output: {1, 2, 3, 4}

my_set.remove(2)
print(my_set)                # Output: {1, 3, 4}

my_set.discard(5)
print(my_set)                # Output: {1, 3, 4}

Set Methods

Sets provide a variety of built-in methods to perform operations like checking membership, finding the length, copying, and more. Some commonly used methods include len(), copy(), isdisjoint(), issubset(), issuperset(), and clear(). 


my_set = {1, 2, 3}

print(len(my_set))           # Output: 3

new_set = my_set.copy()
print(new_set)               # Output: {1, 2, 3}

disjoint = my_set.isdisjoint({4, 5, 6})
print(disjoint)              # Output: True

subset = {1, 2}.issubset(my_set)
print(subset)                # Output: True

my_set.clear()
print(my_set)                # Output: set()
