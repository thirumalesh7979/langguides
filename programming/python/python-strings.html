Python Strings

In Python, a string is a sequence of characters enclosed within single quotes (') or double quotes ("). Strings are one of the most commonly used data types, and Python provides several built-in methods to manipulate and work with strings.

Creating Strings

To create a string, you can simply assign a sequence of characters to a variable using quotes. 

my_string = 'Hello, World!'

Accessing String Characters

You can access individual characters in a string using indexing. Indexing starts at 0 for the first character, and you can use positive or negative indices. 


my_string = 'Hello, World!'

print(my_string[0])      # Output: 'H'
print(my_string[7])      # Output: 'W'
print(my_string[-1])     # Output: '!'
print(my_string[-6])     # Output: 'W'

String Length

You can find the length of a string using the len() function, which returns the number of characters in the string. 

my_string = 'Hello, World!'
print(len(my_string))    # Output: 13

Modifying Strings

Strings in Python are immutable, which means you cannot change individual characters. However, you can create new strings by concatenating or manipulating existing strings. 

my_string = 'Hello'
new_string = my_string + ', World!'
print(new_string)        # Output: 'Hello, World!'

uppercase_string = my_string.upper()
print(uppercase_string)  # Output: 'HELLO'

lowercase_string = new_string.lower()
print(lowercase_string)  # Output: 'hello, world!'
