Python Input and Output

Input and output (I/O) operations are essential for any programming language, including Python. In Python, you can interact with users by taking input from them and displaying output to them. This tutorial will cover the basics of input and output in Python.

Printing Output

You can use the built-in print() function to display output in Python. The print() function accepts one or more arguments and prints them to the standard output (usually the console). 


print("Hello!")
# Output: Hello!

name = "Leo"
age = 25
print("My name is", name, "and I am", age, "years old.")
# Output: My name is Leo and I am 25 years old.

Reading Input

To read input from the user, you can use the input() function. The input() function prompts the user for input and returns the entered value as a string. 


name = input("Enter your name: ")
print("Hello,", name)

age = input("Enter your age: ")      # take input as string by default
# use this method to take a number 
# age = int(input("Enter your age: "))
print("You are", age, "years old.")

Formatting Output

Python provides several ways to format output, including the use of f-strings and the format() method. 


name = "Jake"
age = 25
print(f"My name is {name} and I am {age} years old.")

quantity = 10
price = 5.99
total = quantity * price
print(f"Total cost: ${total:.2f}")

# My name is Jake and I am 25 years old.
# Total cost: $59.90

Using format() method


name = "Jake"
age = 25
print("My name is {} and I am {} years old.".format(name, age))

quantity = 10
price = 5.99
total = quantity * price
print("Total cost: {:.2f}".format(total))

# My name is Jake and I am 25 years old.
# Total cost: 59.90
